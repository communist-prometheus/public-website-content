# Почему выбрать фреймворк Astro

Astro - это современный веб-фреймворк, который привносит свежий подход к созданию веб-сайтов. Вот почему он набирает популярность.

## Преимущества Astro

### Меньше JavaScript

Частичная гидратация Astro означает, что вы отправляете JavaScript только для интерактивных компонентов:

- Статический HTML по умолчанию
- Интерактивные островки при необходимости
- Значительно меньшие размеры бандлов
- Более быстрая загрузка страниц

### Используйте любимые инструменты

Astro не зависит от фреймворка:

- Используйте свой UI фреймворк (React, Vue, Svelte)
- Используйте несколько фреймворков в одном проекте
- Используйте существующие библиотеки компонентов
- Постепенная миграция

### Коллекции контента

Встроенное управление контентом:

```typescript
const posts = await getCollection('blog');
const sortedPosts = posts.sort((a, b) => 
  b.data.pubDate.getTime() - a.data.pubDate.getTime()
);
```

### Типобезопасность

Полная поддержка TypeScript с автоматической генерацией типов для коллекций контента.

## Идеально для

- Маркетинговых сайтов
- Блогов и документации
- Витрин электронной коммерции
- Контент-тяжёлых приложений

## Производительность по умолчанию

Astro оптимизирует автоматически:

- Оптимизация изображений
- Автоматическое разделение кода
- CSS изоляция
- Ноль JavaScript по умолчанию
